# recallary

App for recall words and information

```
Для начала я хочу написать фронтенд. В плане функциональности меня интересует следующее:

ПОльзователь вбивает слово и его перевод (слово на английском перевод на русском) после на сайте начинается режим "упражнения"

Пользователь должен вписать слово по буквам, либо накликать слово из букв, которые высвечиваются

Можно пока всё сохранять локально, меня пока интересует только фронтенд, потом уже буду думать об остальном
```

```
Мне не нравится, что используется вспылвающее окно, когда пользователь ответил правильно. Хотелось бы увидеть что-то вроде подсветки слова, которое было угадано.
И в целом какой-то странный стиль, всё в рамках, нет стиля.
Также хочу возможность перетаскивания букв, чтобы можно было сложить слово из букв.
Также я нашёл баг в том, что есть повторящиеся буквы, но среди доступных букв их нет. Точнее если 1 раз нажмёшь на букву, то она будет недоступна.
Также у нас разные поля для ввода, одно для ввода с клавиатуры, другое для клика по буквам.
```
