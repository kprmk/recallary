# recallary

App for recall words and information

```
Для начала я хочу написать фронтенд. В плане функциональности меня интересует следующее:

ПОльзователь вбивает слово и его перевод (слово на английском перевод на русском) после на сайте начинается режим "упражнения"

Пользователь должен вписать слово по буквам, либо накликать слово из букв, которые высвечиваются

Можно пока всё сохранять локально, меня пока интересует только фронтенд, потом уже буду думать об остальном
```
